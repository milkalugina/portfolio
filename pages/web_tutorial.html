Digital product encyclopedia

An educational project that will introduce you to the tools used in the design of digital products.

Cultural context

The demand for professions in digital has grown by 1561% since 2010. The sphere of popularization of digital professions is very relevant now, and most educational platforms focus specifically on specialties in the digital sphere.

Problems

— Lack of affordable education in this area.
— Services close to the idea are intended for users with a different level of knowledge.
— The focus of similar online textbooks is on analytics and development, not product design.
— Educational platforms do not teach the tools that will be in our web tutorial.

Decision

Interactive web tutorial on digital product design.
— An educational project that focuses on developing the practical skills needed to create high-quality digital products.
— Structured information collected according to the Double Diamond methodology.
— Interactive modules that help you immediately «feel» the tool in operation.

The visual language is based on the work done in the last module: I wanted to observe rigor and encyclopedic style, while adding character to the visual part.

Image of a specialist from the field: strict, erudite, specific, but at the same time friendly.

The main page contains the main facts about the web tutorial, how the project structure is built and what the information collection was based on.

The section page tells about the stage and steps that the product goes through. The fixed block displays the main facts characterizing the process.

This page contains basic information for training: text content, interactive blocks, additional information and a fixed block in which the main facts from the paragraph change by scrolling.
